Yellow Duck Framework 2.0

Release History


This is the overview of the different releases.


Pending for Yellow Duck Framework 2.0 beta 3


  • [done] We are now sponsered by Red Carpet Host
  • Complete documentation.
  • [done] Options to log debugging info and log messages to a file.
  • [done] Much nicer layout for some of the internal index pages and the index page of the examples.
  • [done] Release history instead of the build logs.
  • [done] Updated phpThumb to version 1.4.6.
  • [done] YDFSDirectory::getContents is able to return custom objects now.
  • [done] Removed the build directory as it was not needed anymore.
  • [done] YDDatabaseDriver_mysql is no longer a separate include file, which increases performance (as most people tend to use MySQL? anyway).
  • [done] The API of YDBrowserInfo was updated and it's now part of the YDUtil.php include file.
  • [done] The function enableCache and disableCache were added to the YDTemplate class.
  • [in progress - docs need update] Template class and caching need rethinking. Currently, they don't really work together. On top of that, we should give the user the option to not automatically instantiate the template object when creating the request (e.g. when the user decides not to use the Smarty Light template engine). What I might come up with is that you need to instantiate the YDTemplate object manually when you instantiate your YDRequest based class. The template directory should also be fixed at that point, making caching a lot easier.
  • The Smarty Light libraries will probably be updated to 2.1.3 (when available).
  • Add conditional HTTP to the YDFeedCreator module (description)


18 july 2004 - Yellow Duck Framework 2.0 beta 2 (build 450)


  • Switch from Smarty to Smarty Light for the template engine.
  • PHP 5 compatible, but doesn't yet take advantage of the new features.
  • All features from the underlying template engine are now available in the YDTemplate class.
  • Added constant YD_CRLF which contains the end of line appropriate for the current platform.


15 july 2004 - Yellow Duck Framework 2.0 beta 1 (build 343)


  • First public beta version
